Post-Production

Using video editing applications like iMovie allows you to apply special effects and transformations to your exported QuickTime movie. But what if you want to use effects and filters only available to image editing applications like Adobe Photoshop® or GimpTM? What if you'd like to edit some frames of your movie by hand?
The way FrameByFrame saves movie documents allows you to do all of that. Here's how.

Create and save a document

Create your movie document as you normally would. Then save it to disk, close FrameByFrame and select its file in a Finder window.

Reveal the document's contents

Control-click the file (or right-click it, you Mighty Mousers) and choose "Show package contents" from the context menu. A new window will open, containing a lot of files. These are the images your document contains. They are actually just image files in the TIFF format without filename extension. You can open them normally, e.g. using your Mac's Preview application.

Edit the document's images

You can now use any image editor to alter those files in any way you like. Photoshop, Gimp and many other applications even come with a so-called batch processing feature, which allows you to edit any number of files at once. For example, you may want apply a filter to all of your images to make them look like oil paintings, or to add motion blur, etc.

Continue working with your document

Once all changes have been made and saved, re-open your document in FrameByFrame. You may now export your movie and obtain a regular QuickTime movie containing all the glorious effects modern image editors have to offer.
