Getting started with FrameByFrame

So you've downloaded FrameByFrame - where do you go from here? This guide will help you get your feet wet.

Connect a camera to your Mac

FrameByFrame can use any kind of webcam or other video input device your Mac supports (still image cameras are not supported). Connect a camera via USB or FireWire; if you are going to use your built-in iSight, sit back and do nothing until the next step!

Create a new movie

You will need an empty movie document to hold the pictures you take. To create one, use the Organizer. If the Organizer window isn't visible, choose File > New, or press ⌘-N. Click the New movie tab, select one of the predefined movie resolutions (or enter your own), and press Create movie.

The movie window for your newly created movie will show up, and video feed will start playing after a few seconds. If no video feed appears, or if you want to change the input device, switch to the Input tab and select the camera you want from the box labelled Input device. If the camera you want to use is not listed, visit the macam project which provides drivers for a wide array of third-party cameras.

Take some pictures

It's finally time to take your first picture. Point your camera at something interesting, and click the shutter button. Snap! A new picture has been added to your movie reel. Keep taking pictures, changing your scene bit by bit.

Save or export your movie

Now that you have created an interesting sequence of pictures, you'll want to save your work, so that you can re-open it later and continue working. Use File > Save as... or press ⌘-⇧-S to save your movie document as a FrameByFrame file. You can open it later with FrameByFrame.

If you'd rather create a QuickTime movie that can be played by any movie player, use File > Export movie.... It's as simple as that.
